NAME

Enchant - list provider and dictionary information, and do basic batch spell-checking

SYNOPSIS

enchant-3 [OPTION...] [FILE...]

DESCRIPTION

Enchant gives information about Enchant’s spell-checker providers and the dictionaries each offers. See enchant(5) for details about how the providers and dictionaries are located. It can also list misspelled words in input files or standard input.
-d, --dictionary DICTIONARY

use the given dictionary

-l, --check-spelling

List misspellings in the input files, or standard input if none given.

-p, --pwl WORDLIST

use the given personal wordlist

-L, --show-lines

display line numbers

--list-providers

List spelling providers.

--list-dictionaries

List all dictionaries, and which provider will be used for each.

--default-dictionary

Show the default dictionary for the given or default language.

--word-chars

Show the extra word characters for the given or default language, if available. This is of little interest to most users.

-h, --help

display help and exit

-v, --version

display version information and exit

FILES AND DIRECTORIES

See enchant(5) for details of configuration files and personal word lists.
